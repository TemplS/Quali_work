# Quali_work

Проект Кинопоиск предназначен для поиска, подбора и просмотра фильмов по критериям,
просмотра сериалов, покупки билетов в кинотеатры.
Так же можно зарегистрироваться и собирать любимые фильмы в коллекцию.
Если разбить проект на части, то можно отдельно выделить следующие:
- Авторизация
- Поиск и просмотр фильмов, сериалов, а так же информации о них и актёрах
- Просмотр телеканалов разных жанров
- Покупка билетов

Для запуска автотестов необходимо находится в папке test.
Для запуска автотест ui использовать команду pytest test_ui.py
Для запуска автотест api использовать команду pytest test_api.py
Для запуска всех тестов pytest

В тестах UI применяется WebDriverWait для стабильности тестов, т.к.
в его отсуствие большинстов тестов падает из-за нагружености сайта и не возможности ответить вовремя.
Не знаю с чем это связано, однако на слабых компьютерах может появится необходимость пропысывать
wait на каждый шаг с указанием соответствующего локатора. 

Implicitly time выставлен на 120 секунд, т.к. Кинопоиск защищается себя от ботов и
есть необходимости пройти капчу самостоятельно на UI тестах один раз.
Каждый новый тест по UI будет открываться в новой вкладке. По окончанию браузер закроется.
